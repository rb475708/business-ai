## 三、完整注册使用教程

### 1. 注册账号
1. 访问 [CloudCone 官网](https://bit.ly/Cloudcone)
2. 点击右上角 "Sign Up" 按钮
3. 填写注册信息（邮箱用于接收 root 密码和找回账号）

### 2. 创建VPS
1. 登录后选择 "Compute" → "Create New"
2. 自定义配置：
   - 可调整 CPU/RAM/硬盘容量
   - 固定 IP 数量
3. 选择操作系统并设置主机名
4. 点击 "Deploy Server" 开始部署

### 3. 管理VPS
- **状态查看**：后台首页显示 "Active" 表示部署完成
- **常用操作**：
  - 通过 "Manage" 进入控制面板
  - 修改 root 密码
  - 调整配置/重装系统
  - 添加高防 IP
  - 销毁 VPS
